Набор тестов для проверки работы базовых механизмов алгоритма.

Для запуска тестов выполнить команду
python -m unittest tests/test_base.py

Выполняются следующие тесты:
  1. Формирование отчета в нормальных условиях: есть все необходимые (заданные в конфиге) папки и файлы, есть необработанные файлы логов.
